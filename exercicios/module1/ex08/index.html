principais formatações

destaque

Usando 'destaque' de forma não semantica: está em negrito(usando tags b)

usando destaque com semântica:

está em destaque! é um termo forte(usando a tag STRONG)

itálico

nesta frase temos uma frase em italico não semântico: italico não semântica(usando a tag 'i' → sem semantica)

Com semântica:

está em italico com semantica(usando a tag 'EM' → com semântica)

Texto marcado

podomos criar uma parte de um texto usando a tag <mark>

usando parâmetro style o estilo não é herdado pelos outro mark

texto grande e pequeno?

usando texto grande e texto pequeno

marcando texto excluido e inserido

para usar o texto como excuído usamos a tag del, e o texto inserido usamos a tag ins(insert)

texto sobreescrito

para inserir texto sobreescrito usamos a tag <sup> : x20+3

texto subiscrito

para inserir texto subescrito usamos a tag <sub> : H2O
